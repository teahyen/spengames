## 🎯 구현된 기능

### ✅ 핵심 게임플레이
- [x] **미로 회전 메커니즘**: 플레이어가 캐릭터 대신 미로를 회전
- [x] **물리 엔진**: 실시간 중력, 충돌 감지, 마찰력
- [x] **60개 레벨**: 난이도 점진적 증가 (Easy → Medium → Hard → Expert)
- [x] **승리 조건**: 시작점(노란색)에서 목표점(주황색)까지 이동

### ✅ 레벨 시스템
1. **Easy (레벨 1-15)**: 3x3 ~ 4x4 그리드, 기본 메커니즘 학습
2. **Medium (레벨 16-35)**: 4x4 ~ 5x5 그리드, 장애물 등장
3. **Hard (레벨 36-50)**: 5x5 ~ 6x6 그리드, 시간 제한 추가
4. **Expert (레벨 51-60)**: 6x6 그리드, 시간+이동 제한, 파워업

### ✅ 특수 기능
- **파워업 시스템**:
  - 🕐 시간 추가 (+30초)
  - ⚡ 텔레포트 (랜덤 위치)
  - ❌ 장애물 제거
- **장애물**: 충돌하면 튕겨나감
- **시간 제한**: 고난이도 레벨에 적용
- **이동 횟수 제한**: 전문가 레벨에 적용

### ✅ 조작 방법
- **PC**: 
  - 마우스 드래그
  - 방향키 (← / →)
  - 화면 하단 버튼
- **모바일**:
  - 터치 드래그
  - 자이로 센서 (기기 기울이기)
  - 화면 하단 버튼

### ✅ UI/UX
- **메인 메뉴**: 레벨 선택 그리드 (해금 시스템)
- **게임 화면**: 실시간 정보 표시 (레벨, 이동 횟수, 시간)
- **스테이지 클리어**: 별 평가 시스템 (1-3성)
- **올 클리어**: 모든 레벨 완료 축하 화면
- **반응형 디자인**: 모든 화면 크기 지원

### ✅ 진행 상황 저장
- LocalStorage 기반 자동 저장
- 각 레벨별 최고 기록 저장
- 별 평가 저장
- 레벨 해금 상태 저장

### ✅ PWA 지원
- Service Worker 구현
- Manifest.json 설정
- 오프라인 캐싱
- 홈 화면 추가 가능

### ✅ 추가 기능
- **이스터 에그**: 코나미 코드 (↑↑↓↓←→←→BA) - 모든 레벨 해금
- **키보드 단축키**: R키로 재시작
- **자동 일시정지**: 탭 전환 시
- **음소거 기능**: 사운드 on/off (audio.js)

## 🏗️ 기술 아키텍처

### 파일 구조
```
webapp/
├── index.html              # 메인 HTML
├── manifest.json           # PWA 매니페스트
├── sw.js                   # Service Worker
├── src/
│   ├── css/
│   │   └── main.css       # 스타일 (7KB)
│   ├── js/
│   │   ├── main.js        # 진입점 (5.6KB)
│   │   ├── game.js        # 게임 엔진 (19KB)
│   │   ├── physics.js     # 물리 엔진 (6.2KB)
│   │   ├── levels.js      # 60개 레벨 데이터 (8.7KB)
│   │   ├── ui.js          # UI 컨트롤러 (6.7KB)
│   │   └── audio.js       # 사운드 시스템 (7.1KB)
│   └── assets/
│       ├── images/        # 이미지 리소스
│       └── sounds/        # 사운드 효과
└── README.md              # 프로젝트 문서
```

### 주요 클래스

#### `PhysicsEngine`
- 중력 시뮬레이션
- 충돌 감지 (벽, 장애물)
- 마찰력 적용
- 속도 제한

#### `Game`
- 게임 루프 관리
- 레벨 로딩
- 플레이어 상태 관리
- 입력 처리 (마우스, 터치, 키보드, 자이로)
- 렌더링

#### `UIController`
- 화면 전환
- 레벨 선택 그리드
- 진행 상황 표시
- 이벤트 리스너

## 🎮 게임 플레이 가이드

### 기본 플레이
1. 메인 메뉴에서 레벨 선택
2. 화면을 드래그하거나 버튼을 눌러 미로 회전
3. 중력으로 공이 굴러가도록 유도
4. 노란색에서 주황색까지 이동
5. 벽과 장애물 피하기
6. 파워업 수집하여 도움 받기

### 고득점 팁
- 이동 횟수를 최소화하세요 (3성 획득)
- 파워업을 전략적으로 활용하세요
- 미리 경로를 계획하세요
- 물리 법칙을 이용하세요
