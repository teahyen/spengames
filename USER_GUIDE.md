# 섞고 섞고 돌리고 섞고 - 사용자 가이드

## 🎮 게임 소개

**섞고 섞고 돌리고 섞고**는 미로를 회전시켜 금색 공을 목표 지점까지 이동시키는 물리 기반 퍼즐 게임입니다.

## 🎯 게임 목표

- **노란 공** (플레이어)을 시작 위치에서 **주황색 원** (목표)까지 이동시키세요
- 미로를 90도씩 회전시켜 중력을 이용해 공을 굴리세요
- 벽에 막히지 않고 통로를 통해 이동하세요

## 🕹️ 조작 방법

### 💻 데스크톱 (PC)

1. **키보드**
   - `←` 왼쪽 화살표 또는 `A` 키: 미로를 왼쪽으로 90° 회전
   - `→` 오른쪽 화살표 또는 `D` 키: 미로를 오른쪽으로 90° 회전

2. **마우스**
   - 게임 화면을 클릭하고 좌우로 드래그: 미로 회전
   - 화면 하단의 버튼 클릭: 좌우 회전

### 📱 모바일 (스마트폰/태블릿)

1. **터치**
   - 화면을 터치하고 좌우로 스와이프: 미로 회전
   - 하단 버튼 탭: 좌우 회전

2. **자이로 센서** (선택사항)
   - 기기를 좌우로 기울이면 미로가 회전됩니다
   - iOS에서는 첫 실행 시 "자이로 센서 활성화" 버튼을 눌러 권한을 허용하세요

## 📊 난이도 레벨

게임은 4가지 난이도로 구성된 **60개 레벨**을 제공합니다:

### 🟢 초급 (레벨 1-15)
- 3x3, 4x4 크기의 작은 미로
- 장애물 없음
- 게임 메커니즘 학습

### 🟡 중급 (레벨 16-35)
- 4x4, 5x5 크기의 미로
- 빨간 장애물 등장 (부딪히면 튕겨나감)
- 복잡한 통로

### 🟠 고급 (레벨 36-50)
- 5x5, 6x6 크기의 큰 미로
- 장애물 + **시간 제한**
- 화면 상단의 파란색 바가 시간을 표시

### 🔴 전문가 (레벨 51-60)
- 6x6 크기의 최대 미로
- 장애물 + 시간 제한 + **이동 횟수 제한**
- 최고의 전략이 필요

## ⭐ 별 획득 시스템

각 레벨을 클리어하면 성과에 따라 1~3개의 별을 받습니다:

- ⭐ **1개 별**: 레벨 완료
- ⭐⭐ **2개 별**: 최적 이동 횟수 이내로 완료
- ⭐⭐⭐ **3개 별**: 최적 시간과 이동 횟수로 완료

💡 **팁**: 별 3개를 모으려면 빠르고 효율적으로 플레이하세요!

## 🎲 게임 요소

### 플레이어 (노란 공)
- 중력의 영향을 받아 미로의 경사면을 따라 굴러갑니다
- 벽에 부딪히면 멈춥니다
- 속도가 너무 빠르면 제어가 어려워집니다

### 목표 (주황색 원)
- 플레이어가 도달해야 할 지점
- 반짝이며 펄스 효과가 있습니다
- 플레이어가 닿으면 레벨 클리어!

### 벽 (검은색 선)
- 플레이어의 이동을 막습니다
- 각 타일은 4방향(북, 동, 남, 서) 중 일부만 통로가 있습니다

### 장애물 (빨간 원)
- 중급 이상에서 등장
- 부딪히면 플레이어가 튕겨나갑니다
- 피하거나 조심스럽게 통과하세요

## 📱 메뉴 구성

### 메인 메뉴
- **게임 제목**: "섞고 섞고 돌리고 섞고"
- **시작 버튼**: 레벨 1부터 시작
- **레벨 선택 그리드**: 
  - 숫자: 레벨 번호
  - 녹색 + 별: 완료한 레벨 및 획득 별 개수
  - 자물쇠 🔒: 잠긴 레벨 (이전 레벨을 먼저 클리어하세요)

### 게임 화면
- **상단 바**:
  - 현재 레벨 번호
  - 이동 횟수
  - 재시작 버튼
  - 메뉴로 돌아가기 버튼
- **중앙**: 게임 캔버스
- **하단**: 회전 버튼 (← 왼쪽 / 오른쪽 →)

### 클리어 화면
- 레벨 번호
- 이동 횟수
- 소요 시간
- 획득한 별
- **다음 레벨** 버튼
- **메뉴로** 버튼

## 💡 게임 팁

1. **천천히 시작하세요**
   - 처음에는 미로의 구조를 파악하세요
   - 어느 통로가 목표로 이어지는지 확인하세요

2. **회전 타이밍**
   - 너무 빨리 회전하면 공이 제어하기 어렵습니다
   - 공이 벽에 닿았을 때 회전하면 안전합니다

3. **장애물 피하기**
   - 장애물은 반드시 피할 필요는 없습니다
   - 때로는 장애물에 튕기는 것을 이용할 수 있습니다

4. **경로 계획**
   - 회전하기 전에 공이 어디로 굴러갈지 예측하세요
   - 최단 경로가 항상 최선은 아닙니다

5. **연습이 중요합니다**
   - 실패해도 괜찮습니다! 재시작 버튼을 누르세요
   - 여러 번 시도하면서 최적의 경로를 찾으세요

## 🏆 진행 상황 저장

- 게임은 자동으로 진행 상황을 저장합니다
- 레벨을 클리어하면 별 개수와 함께 저장됩니다
- 브라우저를 닫아도 진행 상황이 유지됩니다
- 같은 브라우저에서 다시 접속하면 이어서 플레이할 수 있습니다

## 🔧 문제 해결

### 게임이 로딩되지 않아요
- 브라우저를 새로고침하세요 (F5 또는 Ctrl+R)
- JavaScript가 활성화되어 있는지 확인하세요
- 최신 브라우저를 사용하세요 (Chrome, Firefox, Safari, Edge)

### 조작이 반응하지 않아요
- 게임 화면을 한 번 클릭하세요
- 페이지가 완전히 로딩될 때까지 기다리세요
- 다른 탭이나 프로그램이 키보드 입력을 가로채지 않는지 확인하세요

### 모바일에서 자이로가 작동하지 않아요
- "자이로 센서 활성화" 버튼을 눌러 권한을 허용하세요
- 일부 브라우저에서는 자이로 센서를 지원하지 않을 수 있습니다
- 터치 조작으로도 충분히 플레이할 수 있습니다

### 진행 상황이 사라졌어요
- 브라우저 쿠키/캐시를 삭제하면 진행 상황이 초기화됩니다
- 브라우저의 "사생활 보호 모드"에서는 저장되지 않습니다
- 다른 브라우저나 기기에서는 진행 상황이 공유되지 않습니다

## 📱 오프라인 플레이

이 게임은 **Progressive Web App (PWA)** 입니다:

1. **모바일에서 홈 화면에 추가**:
   - Chrome: 메뉴 → "홈 화면에 추가"
   - Safari: 공유 → "홈 화면에 추가"

2. **오프라인 플레이**:
   - 한 번 접속하면 인터넷 없이도 플레이 가능
   - 진행 상황은 기기에 저장됩니다
